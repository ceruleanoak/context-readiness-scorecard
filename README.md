# Making AI Work — Tools & Resources

A series of interactive tools for enterprise AI practitioners.

## Tools in this Series

### Part 1: Context Readiness Scorecard
[**→ Open Tool**](https://ceruleanoak.github.io/context-readiness-scorecard/)

Assess your organization's readiness for enterprise AI that delivers ROI. A 5–7 minute diagnostic across four dimensions — Data Integration, Organizational Knowledge, AI Capability Deployment, and Governance & Oversight — with a downloadable results report.

### Part 2: The Context Stack
[**→ Open Tool**](https://ceruleanoak.github.io/context-readiness-scorecard/context-stack.html)

An interactive deep-dive into the four-layer architecture that separates AI that compounds value from AI that silently fails. Import your Part 1 scorecard results to unlock personalized insights and generate an executive readout with a prioritized initiative roadmap.
